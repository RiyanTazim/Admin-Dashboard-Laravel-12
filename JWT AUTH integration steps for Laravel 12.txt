JWT AUTH For Laravel 12 Admin DashBoard

Step 1: composer require tymon/jwt-auth

Step 2: add api.php

Step 3: config/auth.php -> 'guards' => [
    'web' => [
        'driver' => 'session',
        'provider' => 'users',
    ],

    'api' => [
        'driver' => 'jwt',
        'provider' => 'users',
        'hash' => false,
    ],
],

Step 4:'providers' => [
    'users' => [
        'driver' => 'eloquent',
        'model' => App\Models\User::class,
    ],
],

Step 5: php artisan vendor:publish --provider="Tymon\JWTAuth\Providers\LaravelServiceProvider"
php artisan jwt:secret

Step 6: USER model->  implements JWTSubject(beside class User extends Authenticatable)
  use Tymon\JWTAuth\Contracts\JWTSubject; ->
 /**
     * ✅ Required by JWTSubject
     */
    public function getJWTIdentifier()
    {
        return $this->getKey();
    }

    /**
     * ✅ Required by JWTSubject
     */
    public function getJWTCustomClaims()
    {
        return [];
    }

Step 7: Create AuthController and integrate user register, login, logout, reset function for API
